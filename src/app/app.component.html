Model Information

This model was trained on the MNIST dataset, which consists of 60,000 handwritten digits. The model has an input layer of 784 neurons (28x28 pixels), one hidden layer with 20 neurons, and an output layer with 10 neurons (one for each digit).

The model uses the sigmoid activation function for the hidden layer and softmax for the output layer. It was trained over 10 epochs using batch gradient descent with a batch size of 64 and a learning rate of 0.01. The model achieves 94% accuracy on the test set.
